
## Pokedex

Esta es una aplicación que te permite encontrar los detalles de tu Pokemon favorito.

Se utilizó la Api "PokeApi" para obtener los datos.

Cuenta con version móvil.

Para realizar esta web se utilizó:

* Bootstrap 3
* JQuery 3.2.1
* Api PokeApi
